# XUD EXPLORER WEB

A graphical user interface for interacting with a [xud-docker](https://github.com/ExchangeUnion/xud-docker) environment.

## Development

### Requirements

- Node v12.1.0+
- Yarn

### Install dependencies

`yarn`

### Start in development mode

`yarn start`

### Tests

`yarn test`

### Lint

`yarn lint`
